# Publish with The Game Crafter

## Create an account
- Visit [The Game Crafter](https://www.thegamecrafter.com/) and sign up with an email address.
- Confirm the account to unlock publishing and store tools.

## Prepare a game project
- From the dashboard, create a game and pick a product line. Playing cards use the Poker Deck product.
- Add a name, short description, and a package type so the system can price the print run.

## Upload cards from the exporter
- Use **Export to The Game Crafter** in this app and log in through the modal.
- Pick a designer profile and select a game or create a new one.
- Send the deck. Each PNG goes straight into the selected game as a card asset.

## Order a print run
- Open the game on The Game Crafter site and start a print order.
- Pick a quantity, review proof sheets, and submit payment.
- Shipping options appear after the proof approval step.

For help on card layout before export, open [Layout and global options](help:layout-global-options).
