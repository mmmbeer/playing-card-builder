# Rank and index fonts

## Pick a font
- Open the font browser from the **Rank/Index font** panel.
- Search Google Fonts and apply a family to all ranks.
- The label shows the active family.

## Size and weight
- Adjust **Size (px)** for larger or smaller indexes.
- Choose a **Weight** that matches the art style.

## Colors
- **Single color** sets one swatch for every suit.
- **Bi-color (red / black)** splits colors by suit color.
- **Per suit** lets each suit use its own swatch.

## Effects and outlines
- Pick an **Overlay / effect** such as glow or shadow.
- Toggle **Outline / border** and set width, placement, and color.

## Alignment and spacing
- Use the **Rank offset** and **Suit offset** sliders in **Corner spacing** to nudge indexes in small steps.
- Keep text inside the safe zone when using large fonts or heavy outlines.
