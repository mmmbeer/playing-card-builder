# Glossary

| Term | Meaning |
| --- | --- |
| Ability text | The markdown text block that appears near the top or bottom of the card. |
| Bleed | Extra art beyond the trim line. Printers trim here, so keep vital art inside the safe zone. |
| Bulk upload | A batch import that maps many face images to suits and ranks at once. |
| Corner spacing | The controls that shift rank and suit indexes in small increments. |
| Icon sheet | A 2Ã—2 sprite sheet that holds suit symbols. |
| Mirror corners | A toggle that copies the top-left rank and suit into the bottom-right corner. |
| Overlay | A glow or shadow effect applied to fonts or icons. |
| Outline | A stroke around text or icons. Width and placement controls sit near each toggle. |
| Pip | The suit symbol that appears in the center layout and in the corners. |
| Safe zone | The printable interior of the card where text and art stay readable. |
| Suit preset | A built-in icon sheet with ready colors and art. |
| Wild joker | A joker marked as wild to remind players during a hand. |
