# Exporting cards

## Single-card download
- Use the **⬇️** button on the preview bar to download the active card as a PNG.
- The download respects all current settings, including background, fonts, pips, and overlays.

## Full ZIP export
- Click **Export as ZIP File** to build every card in the deck.
- The ZIP groups files by suit and rank. Jokers appear with the chosen label.
- File names list suit and rank for quick sorting.

## Progress tracker
- The progress bar shows rendered cards and time remaining.
- Use the cancel button if you need to stop a long run and change settings.

## The Game Crafter export
- Open **Export to The Game Crafter** to send cards to a linked account.
- Pick a designer, pick an existing game, or create a new one in the modal.
- Uploads honor the same settings used in the preview.
- Read [Publish with The Game Crafter](help:game-crafter) for registration and printing steps.
